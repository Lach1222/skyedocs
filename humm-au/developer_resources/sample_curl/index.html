Sample cURL Script

Below is a sample cURL script that you can use to test the humm test (sandbox) gateway.

curl 'https://securesandbox.shophumm.com.au/Checkout?platform=Default' -H 'Content-Type: application/x-www-form-urlencoded' -X POST -d 'x_account_id=30199250&x_currency=AUD&x_amount=89&x_reference=9051364487&x_shop_country=AU&x_shop_name=Example+Store&x_transaction_type=&x_description=Tim%27s+Pet+Store&x_invoice=239051364487&x_test=true&x_customer_first_name=John&x_customer_last_name=Citizen&x_customer_email=john.citizen%40example.com&x_customer_billing_address1=some+address&x_customer_billing_address2=&x_customer_billing_city=Adelaide&x_customer_billing_state=SA&x_customer_billing_zip=5000&x_url_callback=http%3A%2F%2Fexample.com%2Fcallback.php&x_url_cancel=http%3A%2F%2Fexample.com%2Fcart.php&x_url_complete=http%3A%2F%2Fexample.com%2Fcomplete.php&x_signature=5390d3249610b40b146ada45d46493895dbad0e7ca05daa7df42e6bd0cfc203f'
